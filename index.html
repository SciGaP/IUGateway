SciGaP Campus CyberGateway

This software can be used to deploy Gateways to Campus Cyberinfrastructure. For overview of the software, visit the project website at - http://scigap.github.io/CyberGateway/

The Indiana University Campus Cyberinfrastructure Gateway is powered by this software. For a test ride visit - https://cybergateway.uits.iu.edu/iugateway/.

The code is pretty generic and integrates with CI Logon proxying to InCommon based institutions. The skins to the current version are for IU Gateway but can be easily customized.

If you would like to contribute to the project, please contact the project development mailing list at - http://groups.google.com/a/scigap.org/d/forum/cig-dev. The mailing list is open to public, please click on Join Group or browse on web.

The commit emails from this project will be sent to mailing list - http://groups.google.com/a/scigap.org/d/forum/cig-commits.
